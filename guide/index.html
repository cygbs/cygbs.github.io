快速开始 ​

由于这是一个 Java 版服务器且不兼容基岩版，所以无论您使用何种设备、 何种系统都需要安装 Minecraft: Java Edition。 对此带来的不便，我们非常抱歉。

如果您不想阅读详细步骤，并且您已经打开游戏，请连接：

bash
b2t.xin

安装游戏 ​

使用 HMCL 安装最新版游戏， 这适用于几乎所有桌面端操作系统。

如果您是 Android 或者 iOS 用户， 请参考这个教程吧。

我们暂时不提供基岩版支持。

加入游戏 ​

启动游戏，然后复制上面的地址并加入。
